# Protocol

DeFi protocol,

**Renzo**  基于EigenLayer的再质押协议

是一个基于 EigenLayer 的再质押协议。Renzo抽象了最终用户重质押的复杂流程，质押者不必担心运营商和奖励策略的主动选择和管理。ezETH 是 Renzo 的流动性重质押代币，它会自动获取奖励并确保流动性，ezETH允许用户参与DeFi，同时保留重新抵押奖励。
